# Fly
Игра «Мухобойка» — это аркадная игра на Python с графическим интерфейсом, в которой игрок должен ловить мух с помощью мухобойки.

## 🎮 Игровой процесс

- По экрану летает муха.
- Игрок управляет курсором мыши в виде мухобойки.
- Цель: **ударить по мухе** левой кнопкой мыши, чтобы заработать очко.
- Игра длится **20 секунд**.
- По истечении времени появляется экран с результатом.

## 🖼️ Визуальное оформление

- Муха анимирована (два кадра).
- Мухобойка меняет изображение при ударе.
- Задний фон — комната.
- Игровой интерфейс включает:
  - счётчик очков;
  - таймер обратного отсчёта;
  - экран окончания игры с количеством убитых мух.

## 🕹️ Управление

| Действие | Управление |
|----------|------------|
| Движение мухобойки | Перемещение мыши |
| Удар по мухе | Нажатие левой кнопки мыши |

## ⚙️ Системные требования

- Python 3.x
- Установленные библиотеки:
  - `tkinter` (входит в стандартную поставку Python)
  - `winsound` (только для Windows, используется для звуков)

## 🚀 Запуск игры

1. Убедитесь, что в папке с программой находятся файлы изображений:
   - `комната.png`
   - `муха1.png`
   - `муха2.png`
   - `мухобойка1.png`
   - `мухобойка2.png`

2. Запустите скрипт:
